# 사용자 페르소나

## 고객 페르소나

### P1: 디지털 네이티브 고객 (민지)
- **연령대**: 20~30대
- **디지털 친숙도**: 높음
- **특성**: 스마트폰/태블릿 사용에 익숙, 빠른 주문을 선호
- **동기**: 대기 시간 없이 즉시 주문, 직관적 UI 기대
- **불만 요소**: 느린 반응 속도, 복잡한 단계
- **관련 스토리**: 모든 고객 에픽 (EP-01 ~ EP-04)

### P2: 시니어 고객 (영수)
- **연령대**: 50~60대 이상
- **디지털 친숙도**: 낮음
- **특성**: 태블릿 조작에 서툴고, 큰 글씨와 명확한 버튼 선호
- **동기**: 직원 호출 없이 스스로 주문하고 싶지만 실수가 걱정됨
- **불만 요소**: 작은 버튼, 복잡한 네비게이션, 불명확한 피드백
- **관련 스토리**: 모든 고객 에픽 (EP-01 ~ EP-04) - 특히 접근성 수용 기준에 영향

---

## 관리자 페르소나

### P3: 매장 오너 (준호)
- **역할**: 매장 전체 관리 권한
- **특성**: 매장 설정, 메뉴 관리, 매출 현황 파악에 관심
- **동기**: 효율적인 매장 운영, 메뉴 업데이트, 테이블 초기 설정
- **주요 업무**: 메뉴 등록/수정/삭제, 테이블 초기 설정, 과거 주문 내역 확인
- **관련 스토리**: 모든 관리자 에픽 (EP-05 ~ EP-08)

### P4: 홀 매니저 (수진)
- **역할**: 주문 및 테이블 운영 관리
- **특성**: 실시간 주문 처리와 테이블 회전에 집중
- **동기**: 빠른 주문 확인, 원활한 테이블 관리, 고객 대응
- **주요 업무**: 실시간 주문 모니터링, 주문 상태 변경, 테이블 이용 완료 처리, 주문 삭제
- **관련 스토리**: EP-06 (실시간 주문 모니터링), EP-07 (테이블 관리)
